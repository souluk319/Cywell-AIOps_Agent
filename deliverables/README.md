# Deliverables Index

## Active location

앞으로는 아래 폴더를 기준으로 본다.

```text
deliverables/active/
```

현재 active 산출물:

```text
deliverables/active/Cywell_AI_Sentinel_KOMSCO_Edition_Final_Plan.md
deliverables/active/CAS_v0.1.1_Implementation_Plan.md
```

`Cywell_AI_Sentinel_KOMSCO_Edition_Final_Plan.md`는 제품/구축 기준 계획이고, `CAS_v0.1.1_Implementation_Plan.md`는 v0.1.1 실행 산출물이다. 이전 `Cywell Ops Lens` 계획, 초기 리서치 보고서, 원본 PDF는 참고 재료로만 본다.

## Materials location

최신 산출물에 사용한 재료는 아래 폴더에 분리했다.

```text
deliverables/materials/
```

포함 자료:

- `OpenShift_Lightspeed_커스터마이징_구축_수행_방안.pdf`
- `Cywell_Ops_Lens_Agent_Execution_Plan.md`
- `deep-research-report.md`
- `SOURCE_INDEX.md`

## Working rule

새 작업을 시작할 때는 먼저 active 산출물을 읽고, active 산출물에 충돌이 있거나 근거 확인이 필요한 경우에만 materials를 본다.
